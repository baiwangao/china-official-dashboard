# 领导班子数据导入

把真实公开名单整理为 `leadership.json`，放在这个目录下。页面会自动读取并显示在档案列表顶部。

最小字段：

```json
[
  {
    "id": "unique-id",
    "name": "姓名",
    "region": "地区",
    "system": "地方党政",
    "title": "当前职务",
    "rank": "职级",
    "sources": ["公开来源链接"]
  }
]
```

建议字段：

- `age`: 年龄或出生年份换算后的年龄
- `attention`: `低`、`中`、`高`
- `path`: 履历摘要
- `timeline`: `[["年份", "任职事件"]]`
- `events`: 关联公开事件，先为空数组也可以

真实人物导入时建议只放公开事实；评分和事件风险等分析字段可以后续由爬虫和人工校验流程生成。
